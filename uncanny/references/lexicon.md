# Lexicon: words and phrases

Three tiers. Tier 1 is delete-on-sight. Tier 2 is replace-with-plain. Tier 3 is density-watch: legitimate words that become a signature in combination.

**Anti-rationalisation reminder:** the fix for a Tier 1 phrase is never a synonym of the phrase. "Here's the thing" → "The point is" is the same tell in a new shirt. The fix is stating the content the phrase was announcing.

## Tier 1: delete on sight (the sentence survives without it)

### Throat-clearing openers
- "Here's the thing:" / "Here's what/why/how [X]"
- "Let me be clear" / "Let me explain" / "Let me walk you through"
- "The truth is" / "The uncomfortable truth is" / "The reality is"
- "It turns out that"
- "I'll be honest" / "To be honest" / "Honestly,"
- "Can we talk about"
- "At its core" / "At the end of the day"
- "In today's [fast-paced / digital / competitive] [world / landscape / environment]"
- "In a world where"
- "When it comes to"

### Editorial inserts (the model telling you what matters)
- "It's important to note that" / "It's worth noting that" / "Notably,"
- "It is worth remembering"
- "No discussion of X would be complete without"
- "Interestingly," / "Importantly," / "Crucially," / "Significantly,"
- "This matters because" / "Here's why that matters"

### Emphasis crutches
- "Full stop." / "Period."
- "Let that sink in."
- "Make no mistake"
- "Read that again."

### Meta-commentary and structure narration
- "In this section, we'll" / "The rest of this [document/essay] explains"
- "As we'll see" / "As mentioned above" (just say the thing again briefly, or don't)
- "Plot twist:" / "Spoiler:" / "Hint:"
- "But that's a topic for another [post/day]"

### Chatbot residue (should never survive into a document)
- "I hope this helps" / "Let me know if you'd like"
- "Certainly!" / "Great question" / "Absolutely!"
- "As an AI" / any knowledge-cutoff disclaimer
- Echoes of the prompt ("You asked for a report on X. Here is a report on X:")

### Summary bows
- "In conclusion," / "In summary," / "Overall," / "Ultimately," / "To sum up,"
- Any final paragraph that restates the piece without adding a decision, implication, or next step.

## Tier 2: replace with the plain word

AI reaches for the impressive-sounding option. Reach back.

| AI reach | Plain |
|---|---|
| leverage | use |
| utilise / utilize | use |
| delve into | examine, look at |
| navigate (challenges) | handle, deal with |
| unpack | explain |
| landscape (metaphorical) | market, field, situation |
| ecosystem (non-biological) | market, set of tools, sector |
| tapestry | (delete; name the actual things) |
| testament to | evidence of, shows |
| pivotal | key, central (or name why it matters) |
| underscore | show, confirm |
| foster | build, encourage |
| facilitate | help, enable, run |
| streamline | simplify, speed up |
| elevate | improve, raise |
| robust | reliable, well-tested (or the specific property) |
| seamless | (name what the user doesn't have to do) |
| holistic | complete, whole-of-[X] |
| comprehensive | full, complete (or list what's covered) |
| cutting-edge / state-of-the-art | new, current (or name the capability) |
| game-changer | (name the change) |
| deep dive | analysis, detailed look |
| double down | commit, increase |
| moving forward | from now on, next |
| circle back | return to |
| synergy | (name the combined benefit) |
| stakeholders | (name who: users, the client, the board) |
| boasts (a feature) | has, includes |

| myriad | many |
| plethora | many, plenty |

**Professional-register caveat:** some Tier 2 words are load-bearing in tenders and formal documents ("stakeholders" may be the evaluator's own vocabulary; "comprehensive" may map to a scored requirement). See `registers.md`: mirror the client's terminology where scoring depends on it, and keep the plain word everywhere else.

## Tier 3: density-watch (fine alone, a signature in chorus)

These words co-occur in AI output. One is nothing. Three in a paragraph, or the same one twice on a page, is the tell.

crucial, critical (as generic emphasis), essential, key (adjective), significant, notable, remarkable, fascinating, compelling, powerful, invaluable, seamless, intricate, nuanced, multifaceted, dynamic, transformative, innovative, groundbreaking, profound, rich (metaphorical), diverse, unique, dedicated, passionate, meticulous, thoughtful, actionable, insights, journey (metaphorical), realm, spearhead, champion (verb), empower, unlock, harness, amplify, resonate, align (metaphorical), granular

### Adverb policy (nuanced, not a ban)

Delete **emphasis adverbs**: really, very, actually, literally, genuinely, truly, simply, basically, fundamentally, incredibly, extremely, absolutely, definitely, certainly, undoubtedly, undeniably.

Keep **information adverbs**: they change the claim. "deliberately misled" ≠ "misled". "approximately €2M" ≠ "€2M". "previously undocumented" ≠ "undocumented". "rarely fails" ≠ "fails".

Test: delete the adverb. If the sentence's factual content is unchanged, the deletion stands. If the claim changed, the adverb was doing work; keep it.

### Hedge policy

Real uncertainty gets stated once, precisely: "likely", "we estimate", "in our experience". The tell is stacking: "might potentially, in some cases, somewhat improve" is four hedges on one claim. Collapse to one, or find out and state the fact.

### Lazy extremes

every, always, never, everyone, nobody, all, none, used as rhetorical sweep rather than literal quantifier. If the claim is literally universal, keep it. If it's emphasis, replace with the actual scope ("most teams we've worked with", "in each of the five engagements").
